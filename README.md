# Simple order delivery system

## Overview

This is a simple simulation script that simulate ordering system of a store.

## Features

You can create a person, or a customer, or a driver. A customer can create an order, and the order can be assigned to a driver, the driver then can deliver the package

## How to run

```py
python main.py
```
